Git Code

Open VSCode:

Code: code .

Initialize Git:

Code: git init

Git Status:

Code: git status

Git Add:

Code: git add .

Git Commit:

Code: git commit -m "1st Version"

Git Log:

Code: git log

Git Diff:

Code: git diff index.html

Git Push Work To Remote:

Code:

  • git remote add origin https://github.com/abdullahjahangir/GitCode.git
  • git branch -M main
  • git push -f origin main
