Git Code

Open VSCode:

Code: code .

Initialize Git:

Code: git init

Check Status:

Code: git status

Add Files:

Code: git add .

Commit Files:

Code: git commit -m "1st Version"

Check Log:

Code: git log

Check Diff:

Code: git diff index.html

Push Work To Remote:

Code:

  • git remote add origin https://github.com/abdullahjahangir/GitCode.git
  • git branch -M main
  • git push -f origin main

Pull work From Remote:

Code: git pull -f origin main

Create or Check Branch:

Code:

  • git branch
  • git branch feature_1

Switch Branch Work:

Code: git checkout feature_1

Push Branch:

Code: git push -f origin feature_1

Merge Branch:

Code: git merge feature_1

Delete Branch:

Code: git branch -d feature_1
